# AntiSmurfAnalyser
Este é um analisador de contas para ajudar na detecção de smurfs para a RDC.

## Pré-requisitos
### Node.JS
Antes de começar, certifique-se de ter o Node.js instalado no seu computador. Para instalar o Node.js, acesse [nodejs.org](https://nodejs.org/en/download/prebuilt-installer)..

### Configuração da ApiToken do Stratz
Para utilizar este programa, você precisará de uma ApiToken do Stratz. Siga os passos abaixo para obtê-la:

Acesse [Stratz API](https://stratz.com/api) e faça login ou crie uma nova conta

## Como usar?
Apenas abra o arquivo nomeado de run.bat e siga as orientações

### Download
Baixe o arquivo em .zip [clickando aqui](https://github.com/Arthurmenos/AntiSmurfAnalyser/archive/refs/heads/main.zip)
